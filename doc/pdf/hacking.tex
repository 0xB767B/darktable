\subsection{Disk to Image Data Flow}

Related modules:
\begin{itemize}
  \item imageio \code{dt\_imageio\_*}
  \item db (messily hidden in sqlite3 statements all over the place)
  \item image cache \code{dt\_image\_cache\_*}
  \item mip cache \code{dt\_mipmap\_cache\_*}
  \item image \code{dt\_image\_t}
\end{itemize}

in image cache:
\begin{itemize}
  \item db $\rightarrow$ image struct, trigger imageio disk $\rightarrow$ image entry in db if miss.
\end{itemize}

in mip cache:
\begin{itemize}
  \item db $\rightarrow$ mip[0-4], trigger imageio disk $\rightarrow$ mip4 and mip4 $\rightarrow$ mip[0-3] if miss.
\end{itemize}

in imageio:
\begin{itemize}
  \item[[o]] disk $\rightarrow$ image entry in db, metadata through libraw/magick: \code{dt\_imageio\_import}
  \item[[o]] disk $\rightarrow$ mip4 entry (thumbnail \code{dt\_imageio\_open\_preview}).
  \item[[o]] mip4 $\rightarrow$ mipf (by guessed reverse gamma) \code{dt\_image\_raw\_to\_preview}.
  \item[[x]] full pixels $\rightarrow$ mipf (downscaling) \code{dt\_image\_raw\_to\_preview}.
  \item[[x]] mip4 $\rightarrow$ mip[0-3] \code{dt\_image\_update\_mipmaps}.
\end{itemize}
